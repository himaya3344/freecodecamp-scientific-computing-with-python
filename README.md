# freeCodeCamp Data Analysis with Python Projects

This repository contains my completed projects for the **freeCodeCamp Data Analysis with Python** certification.
The projects focus on Python, Pandas, NumPy, Matplotlib, Seaborn, and SciPy for data analysis, visualization, and statistical prediction.

## Projects Included

| Project                                     | Description                                                                                                       | Folder                                                                                       |
| ------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| Mean-Variance-Standard Deviation Calculator | Uses NumPy to calculate mean, variance, standard deviation, max, min, and sum for a 3x3 matrix.                   | [mean-variance-standard-deviation-calculator](./mean-variance-standard-deviation-calculator) |
| Demographic Data Analyzer                   | Uses Pandas to analyze demographic data from the 1994 Census dataset.                                             | [demographic-data-analyzer](./demographic-data-analyzer)                                     |
| Medical Data Visualizer                     | Uses Pandas, Seaborn, and Matplotlib to visualize medical examination data using categorical plots and heat maps. | [medical-data-visualizer](./medical-data-visualizer)                                         |
| Page View Time Series Visualizer            | Uses Pandas, Matplotlib, and Seaborn to visualize freeCodeCamp forum page views using line, bar, and box plots.   | [page-view-time-series-visualizer](./page-view-time-series-visualizer)                       |
| Sea Level Predictor                         | Uses Pandas, Matplotlib, and SciPy linear regression to predict sea level rise through 2050.                      | [sea-level-predictor](./sea-level-predictor)                                                 |

## How to Run the Projects

Clone the repository:

```bash
git clone https://github.com/himaya3344/freecodecamp-scientific-computing-with-python.git
```

Go into the project folder:

```bash
cd freecodecamp-scientific-computing-with-python
```

Create and activate a virtual environment:

```bash
python -m venv venv
```

For Windows PowerShell:

```bash
venv\Scripts\Activate.ps1
```

Install the required libraries:

```bash
pip install numpy pandas matplotlib seaborn scipy
```

Open any project folder and run its `main.py` file:

```bash
cd sea-level-predictor
python main.py
```
